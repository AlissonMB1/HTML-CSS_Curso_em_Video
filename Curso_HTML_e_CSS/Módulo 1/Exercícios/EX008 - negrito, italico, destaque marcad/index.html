Principais formatações

Negrito / Destaque

Nesssa frase temos um termo em negrito usando a tag <b> (não semântica e sim forma)

nesta frase tenho um termo em destaque usando a tag strong (semântica)

Itálico / Ênfase

Nesta frase, temos um um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em destaque usando a tag <em>(semântica = significado.).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande com a tag <big>e um texto pequeno com a tag <small>nesta parágrafo. Mas lembe-se a tag <big> está obsoleta.

Texto excluído

Podemos marcar um texto como excluído com a tag <del>para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado, com a tag <tag>

Existe também o sublinhado com a <u> (não semântica).

Texto sobrescrito

Para inseri coisa do tipo x20+3, usa a tag <sup>

Texto subescrito

Para inserir coisas do tipo H2O, usa a tag <sub>
